# 忘记 Mac 开机密码如何无损恢复数据？

当你或你的客户忘记了 Mac 的登录密码，且里面有重要数据绝不能抹掉硬盘时，可以尝试以下几种苹果官方的无损重置密码方法。

> 💡 **给客户的白话版解释：什么是 Apple ID？**
> 很多客户会把“电脑开机密码”和“Apple ID 密码”混淆。你可以这样跟他们解释：
> *"Apple ID 就像是您在苹果世界的**‘身份证’**。它通常是一个邮箱地址（比如 `@icloud.com`、`@qq.com`）或者手机号。您当初买这台电脑第一次开机激活时，或者在手机上下载软件时，用来注册和登录的那个账号就是 Apple ID。只要您还记得这个‘身份证’和它的密码，我们就能通过它来证明这台电脑是您的，从而帮您把**开机密码**重新改过来。"*

## 前提条件与芯片区分（极度重要！）

在执行系统级密码恢复前，请务必先确认该 Mac 搭载的是 **Intel 芯片** 还是 **Apple 芯片 (M1/M2/M3等)**，因为它们进入 macOS 恢复模式 (Recovery) 的操作完全不同。
- **Intel 芯片 Mac**：按一下电源键开机，**同时立刻长按 `Command (⌘)` + `R` 键**，直至看到 Apple 标志。
- **Apple 芯片 Mac**：完全关机后，**长按电源按钮**不放，直到屏幕出现“正在加载启动选项”，然后点击“选项”并点“继续”。

---

## 方法一：通过 Apple ID 重置（最简单快捷）

如果该 Mac 曾经绑定过 Apple ID，并在设置中开启了“允许用户使用 Apple ID 重设密码”功能，这是最安全、最推荐的方法：

1. 在 Mac 登录屏幕上，连续输入最多三次任意密码。
2. 直到密码框下方出现提示：**使用您的 Apple ID 重设密码**（或者出现一个带有向下箭头的问号图标）。
3. 点击该提示，输入绑定的 Apple ID 账号和密码。
4. 按照屏幕上的说明创建新密码。
> **⚠️ 官方警告**：使用此方法重设密码后，你的 Mac 会创建一个新的默认登录钥匙串。系统可能会提示你更新或输入旧的钥匙串密码（以便解锁之前保存的网页密码等）。如果你不记得旧密码，选择创建新钥匙串即可，这**不会**导致你硬盘里的文稿、照片等文件丢失。

---

## 方法二：进入恢复模式 (Recovery) 使用“重设密码”助理

如果方法一未生效，可以使用 macOS 恢复功能强制调出重设向导。

### 步骤 1：进入 macOS 恢复模式
根据上述“前提条件”中的指引，根据芯片类型进入恢复模式，直到屏幕显示出拥有您的用户账户和“忘记密码了吗？”的界面，或者直接进入「macOS 实用工具」窗口。

### 步骤 2：打开终端重设密码
1. 在「macOS 实用工具」界面，**不要**点击中间的任何选项。
2. 将鼠标移到屏幕**最顶部**的菜单栏，点击 **实用工具** -> **终端**。
3. 在弹出的终端窗口中，输入这串没有空格的代码，并按回车键：
   ```bash
   resetpassword
   ```
4. 此时，屏幕上会弹出一个官方的**“重设密码”**助理窗口。
5. 按照屏幕上的说明选择管理员账户并设置新密码。完成后，从顶部菜单选取重启电脑，即可使用新密码登录。

---

## ⚠️ 终极阻碍：FileVault 磁盘加密或激活锁

如果在上述过程中，Mac 提示无法重设密码，或要求输入先前的密码来挂载磁盘，这通常是因为开启了 **FileVault（文件保险箱）** 硬件级加密或 **激活锁**。

- 这种情况下，你必须输入当时开启 FileVault 时生成的**恢复密钥**，或者这台电脑绑定的 Apple ID 密码才能解锁磁盘。
- 出于 Apple 对用户隐私和数据安全的最高级保护机制，**如果既没有旧密码，也没有恢复密钥，也没有 Apple ID，硬盘内的数据将绝对无法被任何人（包括 Apple 工程师）提取，你只能选择抹掉 Mac 并重新安装 macOS。**
