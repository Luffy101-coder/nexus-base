## 💡 使用指南

1. **左侧导航**：你的所有设备分门别类，点击展开查看对应知识点。
2. **全局搜索**：不论你在哪个页面，顶部的搜索框都能**极速模糊匹配**你曾经记录过的只言片语。
3. **极简维护**：新增文章只需新建 `.md` 文件，并在 `_sidebar.md` 里添加一条链接，网站就会自动更新。
