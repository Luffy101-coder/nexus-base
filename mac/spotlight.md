# 聚焦搜索 (Spotlight) 高级用法

> 💡 **效率小贴士**：快捷键默认为 `Command (⌘) + 空格键`。

Mac 自带的聚焦搜索不仅可以用来打开应用，还可以进行很多高级操作，极大提升日常使用效率。

## 🧮 1. 快速单位换算与计算

无需打开计算器，直接在输入框中输入算式或单位：

- 输入 `100 usd`，会自动转换为你所在地的货币（如人民币）。
- 输入 `150 lbs` 会转换为公斤。
- 输入 `(100 + 20) * 5`，直接得出结果 **600**。

*📌 选中的结果可以直接按下 `Command + C` 复制结果，无需点开。*

## 📂 2. 按文件类型精确查找

当你只记得文件内容或类型，忘记名字时，可以使用类型过滤词：

```bash
kind:pdf 年度报告
kind:image 旅行截图
kind:folder 项目总结
```

这样搜索可以滤除掉所有无关的应用和文档，直达目标。

> ⚠️ 注意：如果搜索不到文件，请前往 **系统设置 > 聚焦** 中检查是否取消勾选了某些类别的搜索权限。

## 🌤️ 3. 直接查询天气或航班

输入你的航班号（如 `MU5101`），Spotlight 会直接显示航班动态、航站楼及行李提取转盘信息。输入“北京 天气”也能直接查看多日天气预报图表。
